Flex-Grow Property

This property determines how much space a flex item should consume in a flex container. This property can accept positive numeric values and zero. When used, this property will expand the flex item and will increase the size of the item by the factor indicated by the numeric value. This property can be used with two other properties in a shorthand property called "flex" that we will discuss on another page.

Below, are the four scenarios that we will look at on this page. They are as follows:

  • Flex-Grow Default
  • Flex-Grow One Item
  • Flex-Grow All Items Same
  • Flex-Grow All Items Different

Flex-Grow Default

This property sets the factor of growth of the specified item in relation to the other items in the container. By default this value is set to 0. When the size of the container grows, the default behavior of the items is to not grow with the container. The flex items will remain the same after they pass above the initial size of each item.

Flex-Grow Single Value

If we want the first flex item to grow, we need to determine by how much. This will be a number value. This could be numeric, em, rem, or % value. In this example, setting the flex-grow to a value of 1 while the rest of the items have a flex-grow value of 0, will cause the first item to increase its size to fill the remaining space in the flex-container. As the page shrinks, the items will shrink proportionally until the first item reaches the preset size of 100px (defined in the style.css file) and then all items will shrink at the same rate or factor.

Justify-Content Property

The justify-content property aligns flex items along the main axis. if flex-direction is set to row, then the items will move left to right. If the flex-direction is set to column, then the items will move top to bottom.

If we use flex-direction value of row or row-reverse, the main axis is left to right.

If we use the flex-direction value of column or column0-reverse, the main axis is top to bottom.

The available properties are as follows:

  • Flex-start
  • Flex-end
  • Center
  • Space-around
  • Space-between
  • Space-evenly

Flex-start Value

This is the default behavior of the justify-content property when used within the flexbox container. The items start at the left and move from left to right to form a "row". This row will be flush with the left margin of the container.

Space-around Value

This value of the justify-content property when used within the flexbox container will space the items across the container. The first and last items will be flush with the left and right edges of the container and the rest of the items will be spaced evenly in between them. The items will start at the left and move from left to right to form a "row".

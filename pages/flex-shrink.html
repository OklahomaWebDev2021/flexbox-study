Flex-Shrink Property

The flex-shrink property is another of the flex sizing properties. It shrinks the flex items from their preset sizes as the parent container shrinks down on different devices where it is viewed. The flex-shrink can accept as valid values zero and all positive values. It cannot accept negative values. The default for this property is true or on. So, there is no need to set this in the CSS file unless it is being turned off.This must be set inside of each flex item where it is to be changed.

  • Default Value
  • Zero Value
  • Different Values

Default Value

Unlike the flex-grow property, the flex-shrink property is defaulted to a value of 1. This means that the flex items will shrink at the same rate as the device gets smaller.

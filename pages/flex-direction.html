Flex-Direction Property

The flex-direction is which way the flex items will flow. It column is selected, the flex items will go down the screen as they normally would without using the flexbox system. There are four values we will discuss out of six values. Maybe at some point I will come back and discuss the other two, but not right now.

They are as follows:

  • Row
  • Reverse-Row
  • Column
  • Reverse-Column

Row Value

This is the default behavior of the flex-direction property when used within the flexbox container. The items start at the left and move from left to right to form a "row".

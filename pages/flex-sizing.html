Flexbox Sizing Principle

Flexbox main axis sizing has a number of layers of consideration when determining the criteria for the width. In looking a the image below we see how flexbox determines what width to use when sizing a flex item.

  • If no widths are set on a flex item, flexbox will default to the content width.
  • If the width property is used, then flexbox will use this over the content width.
  • If the flex-basis is set, then content width and width property will be ignored.

NOTE: The settings for flex-grow and flex-shrink can have an affect on the sizing of the flex items. The default is flex-grow set to 0 and flex-shrink set to 1 with flex-basis set to auto.

Content Width

Content width looks literally at the content or text typed inside the element. If no other width property is set, then the flex items will base their widths on that content.

In the example below, we see that the length of the length of the flex item along the main axis is snug against the text. As long as all of the text will fit on a single line on the remaining space of the container, that will be the maximum size. As the device gets smaller or the screen shrinks, the text will wrap down to the longest word. This will be the minimum length of the flex item. After that, the flex items will begin to move off the screen.

TIP: If using a laptop, adjusting the size of the browser should show the content width in action. For a tablet or smart phone, try rotating the device to see the length affect.

Min-Width Property

We can also set the min-width for each flex-item. This will affect all three width properties discussed above. This will have different affects on a flex-item based on any preset widths. The min-width determines the shrink potential of each flex item in which it is set.

Min-Width Smaller

If the min-width is smaller than the width or flex-basis properties, then flexbox will use the width or flex-basis property.

Min-Width Larger

If the min-width is larger than the width or flex-basis properties, then flexbox will use the min-width over both the width or flex-basis properties.

Max-Width Smaller

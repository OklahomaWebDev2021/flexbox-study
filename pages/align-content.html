Align-Content Property

The align-content property is used to align flex items inside of the container along the cross axis (see align-items for a diagram of main and cross axis here). This property works on the spaces between the multiple lines of a flex-wrapped set of flex items.

NOTE: This will only work if the flex-wrap property is set to a value of "wrap" and there are multiple lines. The default value for flex-wrap is set to "nowrap".

They are as follows:

  • Stretch
  • Flex-start
  • Flex-end
  • Center
  • Space-Around
  • Space-Between
  • Space-Evenly

Stretch Property

This value is the default value for the align-content property. It will move flex items to the start of the flex container. In flex direction of row or row=reverse, this is the upper left. It also looks like the flex items stretch to fill the cross-axis of the container.

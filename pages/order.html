Order Property

The order property is a flex item level property. It can take a numeric value from a negative to a positve including zero. This will position the individual flex item within the container accordingly. A negative number moves the item to the left and a positive number moves it to the right.

The available properties are as follows:

  • Zero Value
  • Positive
  • Negative

Zero Value

This is the natural order of the flex items. It is the order in which they are entered into the HTML file. So, the first way to control the order of the items is to simply place them into the HTML in the order in which they should appear to those that view your page. No code is needed for this order.

Positive Value

By using a positive number, the item where the order property is used will move to the right. How far to the right depends on if there are other positive values used for other items or if their values are all zero.

In the example below, item 2 and item 3 both have an order value of 0 and item 1 has an order value of 1. This will put item 1 at the end of the items because a value of 1 is the largest positive value.

Negative Value

By using a negative number, the item where the order property is used will move to the left. How far to the left depends on if there are other negative values used for other items or if their values are all zero.

In the example below, item 1 and item 2 both have an order value of 0 and item 3 has an order value of -1. This will put item 3 at the start of the items because a value of -1 is the smaller than either a positive or zero value.

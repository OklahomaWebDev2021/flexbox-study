Flex-Wrap Property

The flex-wrap property will tell flexbox how to handle flex items that move off of the screen in a single row. This is seen when we have responsive design and a webpage designed for the desktop or laptop is viewed on a tablet or phone. Using this property will move the offending items to a line where they will fit.

They are as follows:

  • Nowrap
  • Wrap
  • Wrap-reverse

NOTE: If the items are not flexible, the flex-wrap property will not work.

Nowrap Value

This is the default behavior of the flex-wrap property when used within the flexbox container. The items start at the left and move from left to right.

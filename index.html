What is Flexbox?

Flexbox is a CSS 3 web layout module. With it, the web developer can layout the webpage in a myriad of creative ways.

Flexbox helps to create responsive design webpages without having to use float and positioning.

Flexbox Elements

Within the flexbox layout module there are two main components. These two components are as follows:

  • Container
  • Items

The Container

The container does exactly what its name implies, it contains some kind of HTML element or elements.

To use this to create a responsive layout on the website, we use the display property of CSS. This property is also used for other layout types that can be found here

To see these container properties, please click here.

The Items

The items are components that are one level inside the container or direct children of the container element.

These are the elements that will be formatted and become the layout of the container object.

To see these flex item properties, please click here.
